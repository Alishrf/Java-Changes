# Java Evolution: Major Features from Java 8 to Java 21

## 📖 Overview

This repository demonstrates the most important features introduced in each Java release, starting from **Java 8** up to **Java 21 (LTS)**.

The goal of this project is to provide developers with practical, easy-to-understand examples of the language and platform improvements introduced in every major Java version.

Each feature is implemented with a minimal, self-contained example to help developers quickly understand when and how to use it.

---

## 🚀 Purpose

This repository was created to:

* Learn the evolution of the Java language.
* Understand the most significant features introduced in each release.
* Provide practical code examples instead of only theoretical explanations.
* Help developers prepare for technical interviews.
* Serve as a quick reference for modern Java development.

---

## 📌 Covered Java Versions

| Java Version  | Highlights                                                                             |
| ------------- | -------------------------------------------------------------------------------------- |
| Java 8        | Lambda Expressions, Stream API, Optional, Method References, Default Methods           |
| Java 9        | Module System, Collection Factory Methods, Stream Improvements                         |
| Java 10       | Local Variable Type Inference (`var`)                                                  |
| Java 11 (LTS) | HTTP Client, New String APIs, Lambda Improvements                                      |
| Java 12       | Switch Expressions (Preview)                                                           |
| Java 13       | Text Blocks (Preview)                                                                  |
| Java 14       | Records (Preview), Helpful NullPointerExceptions                                       |
| Java 15       | Text Blocks (Final)                                                                    |
| Java 16       | Records (Final), Pattern Matching for `instanceof`                                     |
| Java 17 (LTS) | Sealed Classes, Enhanced Random API                                                    |
| Java 18       | Simple Web Server                                                                      |
| Java 19       | Virtual Threads (Preview)                                                              |
| Java 20       | Record Patterns (Preview)                                                              |
| Java 21 (LTS) | Virtual Threads, Pattern Matching for `switch`, Record Patterns, Sequenced Collections |

Select the desired Java version and run the corresponding example class.
